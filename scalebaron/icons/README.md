# Button Icons

Place your custom button icons in this folder.

## Icon Specifications

- **Size**: 24x24 pixels (recommended) or 32x32 pixels
- **Format**: PNG with transparency (RGBA)
- **Background**: Transparent (alpha channel)
- **Color**: Your choice (works best with simple, high-contrast designs)

## Icon File Names

Name your icons with these exact filenames:

- `summarize.png` - For "Summarize Data" button
- `preview.png` - For "Preview Composite" button  
- `add_label.png` - For "Add Element Label" button
- `save.png` - For "Save Composite" button
- `batch.png` - For "Process All Elements" button
- `progress.png` - For "View Progress Table" button

## Design Tips

- Keep designs simple and recognizable at small sizes
- Use high contrast colors
- Test at 24x24 to ensure details are visible
- Consider using outlines/strokes for better visibility
- Office-style flat icons work well

## Example Workflow

1. Design icons in your preferred graphics software (Illustrator, Inkscape, etc.)
2. Export as PNG with transparency
3. Resize to 24x24 or 32x32 pixels
4. Save to this folder with the exact filenames above
5. The application will automatically load and use them
